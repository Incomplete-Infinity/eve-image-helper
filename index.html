EVE Image Resolver

What This Does

Introduction

There are an extremely large quantity of Eve Online entities(Characters, Inventory Types, Corporations, Alliances, etc.).

The game tracks all of these by with a very long list containing all of them. This list is indexed by ID.

Supported ID Categories
  • Character
  • Corporation
  • Alliance
  • Faction (NPC Corp)
  • Type (Item)
  • Blueprint
  • Ancient Relic

This tool takes an EVE Online ID and determines which image formats are available using basic ESI lookups. It supports a wide range of in-game entities, which are all listed to the right.

The script first resolves the category (e.g. character, corporation, inventory_type) by querying the universe/names/ endpoint with the given ID. Below, is an reproduction of an actual response to one such query.

Then, it determines the appropriate image variants. For inventory types, it performs additional lookups to distinguish between ships, blueprints, and relics specifically â€” ensuring the most accurate URL formation.

Ships will return both icon and render images. Blueprints return bp and bpc formats. Ancient relics return a relic image. Other IDs resolve to portraits or logos as appropriate.
